# JavaScript Basics

This directory contains examples of JavaScript basic concepts organized by chapter.

## Directory Structure

- `chapter1_data_types.js` - Examples demonstrating JavaScript data types
- `chapter2_operators.js` - Examples of JavaScript operators and their usage
- `main.js` - Original file containing all examples (kept for reference)

## Chapter 1: Data Types

JavaScript primitive data types covered:
- Number: For numeric values (integers and floating-point)
- String: For text values
- Boolean: For true/false values
- Undefined: Variables declared but not assigned
- Null: Represents the intentional absence of any value

Variable declaration keywords:
- `var`: Function-scoped, can be redeclared
- `let`: Block-scoped, cannot be redeclared

## Chapter 2: Operators

JavaScript operators covered:
- Arithmetic operators: +, -, *, /, %, **
- Assignment operators: =, +=, -=, *=, /=, %=, **=
- Logical operators: &&, ||, !
- Bitwise operators: &, |, ^, <<, >>

## How to Run Examples

You can run any example file using Node.js:

```bash
node chapter1_data_types.js
node chapter2_operators.js
```

## Notes

- The examples demonstrate basic concepts and are intended for educational purposes
- Each file includes detailed comments to explain the code
- The `typeof` operator is used to show the data type of variables
