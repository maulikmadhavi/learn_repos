# Introduction
This repository contains few example materials for pandas library in Python. The material is heavily based on YouTube tutorial by "CodeBro" (https://www.youtube.com/watch?v=VXtjG_GzO7Q).
It is intended for basic learning and practice of pandas library. 

# Use cases
- Pandas series creation and manipulation
- DataFrame creation and manipulation
- Data selection and filtering
- Aggregation and grouping
- Filtering 
- Data cleaning 

# Dependencies
Run using pixi. `pixi install` to install dependencies.
